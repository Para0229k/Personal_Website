Programming

我常用的自學網站及上過的一些課程

Coursera

起初是因為學校的雙語政策才上到這個網站的課程，後來自己探索才發現上面有許多優秀的課程可以上，重點是內容不像網路上許多濫竽充數的課程，只是 讓學生花錢買一堆不是自己的作品，而不重視思考過程，更遑論理解原理；Coursera上的課程給我的印象是更像是一般大學教育風格，相當重視思考過程， 比較像是提供資源來讓學生吸收、並給予充足的練習來磨練實作，沒有花里胡俏的噱頭和華而不實的高階作品，而是幫助學生奠定扎實的基礎。

但缺點當然還是有的，平台上大部分的課程都不是免費的，但還是可以透過「旁聽」的選項看上課影片，也可以查看練習的題目及作業，只是不能上傳作業 及回答題目以拿到課程證書。除此之外，Coursera上的課程數量眾多，需要自己篩選高品質的課程來上，不然還是一樣會浪費時間。

上過的課程：

  1. Introduction to Programming with MATLAB (Vanderbilt University)
  2. Mastering Programming with MATLAB (Vanderbilt University)
  3. Introduction to Data, Signal, and Image Analysis with MATLAB (Vanderbilt University)
  4. Divide and Conquer, Sorting and Searching, and Randomized Algorithms (Stanford University)
  5. Graph Search, Shortest Path, and Data Structures (Stanford University)
  6. Introduction to Computer Vision (MathWorks)

Udemy

比較近期才發現的網站，同樣是以英文為主，課程種類相當多元，上課的人數也普遍較Coursera多，附帶的是課程評價較多，比較容易做事前功課，此外， 課程的內容也很豐富。

由於我在這個平台學習的時間還不長，無法做出完整的分析，目前發現最直觀的缺點就是幾乎每門課都是真的要花錢才能上課(還是有免費的課程)，需要做好 事前作業避免花錢買到不是自己要的東西。

上過的課程：

  1. Become a Full-Stack Web Developer with just ONE course. HTML, CSS, Javascript, Node, React, PostgreSQL, Web3 and DApps

Reading

除了小說以外，我也喜歡看一些稍微正經一點的書來檢視我的生活及思考方式

只有勝利，才是我的存在意義！：東大畢業世界電競冠軍Tokido從電玩學會的人生成功之道

Tokido選手是我從小到大的偶像，他出的這本書對我而言意義非凡，雖然我對成為電競選手沒有興趣，但他對生活的熱情和對勝利的執著我成長過程的影響 很大。我每過一個人生階段就會拿出來重新看一次，每次都讓我有不同的體悟，最近一次是在大一的暑假，我發現了另一件Tokido選手值得我學習的地方： 隱藏在火熱的衝動背後的冷靜思考與縝密規劃，在向前衝的同時卻保持頭腦的冷靜，不被熱情與衝動支配，這個發現和另一本書一起給了我很大的衝擊，讓 我清醒過來並意識到自己現在的處境。

連結

大學不知道：別讓大學放榜那一天，成為你人生最高峰。那些畢業後海放同學的人，都怎麼成長？

這本書在大學時徹底改變了我的想法，讓我重新審視了一遍自己，發現自己似乎正在「死於安樂」的路上，並決心拋棄過去的自己，面對更多的挑戰
